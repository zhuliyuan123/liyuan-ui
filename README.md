# liyuan-ui
```
笠源自己实践中的实现插件
```

### 安装
```
$ npm install liyuan-ui -S
```

### 使用

在 main.js 文件中引入插件并注册
```
# main.js
import liyuanUi from 'liyuan-ui'
Vue.use(liyuanUi)
```
在项目中使用 vcolorpicker

```
<li-foldturn>
    <img src="" alt=""> 
</li-foldturn>

```
### 描述

实现图片折叠跑马灯的效果

### 选项

标签下插入图片必须添加img标签
